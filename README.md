## Featured Projects

### Low-Cost PPG-to-ECG Reconstruction System Using TCN

Documentation-only portfolio repository for an ongoing proprietary healthtech research project.

This project explores ECG-like waveform reconstruction from continuous PPG signals using a low-cost wearable prototype and a Temporal Convolutional Network based reconstruction pipeline.

Key work:

* Low-cost PPG acquisition workflow using MAX30102 and ESP32
* PPG signal preprocessing and derivative-based input construction
* TCN encoder-decoder based ECG-like waveform reconstruction
* Benchmark PPG-ECG evaluation
* Custom wearable signal transfer observation
* Web-based visualization and warning-oriented interface planning
* Code, model, firmware, and datasets withheld due to startup/IP confidentiality

Repository: `Low-Cost-PPG-to-ECG-Monitoring-System`

---

### Global Lookahead Pruning for Transformer Language Models

Research-code repository for activation-aware pruning of transformer-based language models.

This project introduces a Global Lookahead Pruning method that combines weight magnitude with future-layer activation differences to estimate weight importance.

Key work:

* Implemented custom post-training pruning workflow
* Compared against global magnitude and layerwise magnitude pruning
* Evaluated sparsity and perplexity trade-offs
* Tested on GPT-2, Qwen-2.5B, and DeepSeekR1
* Added fine-tuning workflow to recover pruned model performance
* Organized code into reusable `src/` modules and reproducible scripts

Repository: `global-lookahead-pruning`

### Region-Specific Ground-Based Drone Detection Using YOLO11

Ongoing computer vision research project studying public-to-local domain shift in drone detection.

The current stage trains a YOLO11 detector on a public drone dataset and evaluates its transfer behavior on locally collected drone videos.

Key work:

* Trained YOLO11 drone detection baseline
* Evaluated public dataset performance using precision, recall, and mAP
* Tested public-trained model on 11 local drone clips
* Identified false positives on humans, windows, branches, poles, and building edges
* Documented public-to-local deployment gap
* Planned future local annotation evaluation and region-specific fine-tuning

Repository: `region-specific-drone-detection-yolo11`
